实验器材:
	STM32F103RET6最小系统板
	超声波风向风速传感器
	风向传感器（用于检测水流流向）
	流速霍尔传感器（检测水流流速）
实验目的:
	利用485转TTL接收超声波风向风速传感器数据并将数据帧截取并解析转换成十进制数据。
	利用485转TTL接收风向传感器的风流向数据
	利用流速霍尔传感器接收水流流速数据
	最终数据汇总通过串口1（433通信）发送到船上
	
硬件资源:
	1,串口2（传感器默认波特率：4800，PA2(TX)/PA3(RX)）
	2,串口1(波特率:115200,PA9(TX)/PA10(RX))
	3,串口3（传感器默认波特率：9600，PB10(TX)/PB11(RX)）
	4,输入捕获（定时器5(TIM5),TIM5的通道1(TIM5_CH1,连接在PA0上面)）
实验现象:
	本实验,STM32通过串口1和上位机对话，STM32通过串口2串口3不断发送数据请求帧到传感器
	传感器返回数据帧到串口2串口3，串口1打印10进制风速风向数据、水流流速流向数据到串口显示。
	
注意事项:
	1,电脑端串口调试助手波特率必须是115200.
	2,请使用XCOM/SSCOM串口调试助手,其他串口助手可能控制DTR/RTS导致MCU复位/程序不运行
